Margin vs Padding

Margin : The use of margins is to create space around elements outside of any defined borders.

Padding : The amount of padding within an element refers to how far away it wants to be from itself.When padding is enabled, the element's size is increased in order to accommodate the gap.

Margin size is fully customizable with CSS. Each of the four sides of an element has its own margin property (top, right, bottom, and left). Different Types of Margins :

  • margin-bottom
  • margin-top
  • margin-left
  • margin-right

Use padding when You don't want your content to touch the container's edges. Different Types of Padding :

  • padding-bottom
  • padding-top
  • padding-left
  • padding-right

Ways to use Margin:

  • You want to center an element. If you give “margin: auto” to an element with fixed width, it will center that element horizontally
  • The element needs to be surrounded by some space, or it doesn't want to touch any other elements nearby
